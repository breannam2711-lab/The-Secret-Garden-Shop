# Flower Shop Website 🌸

This is a flower shop website project created for a school assignment. It is built using **HTML, CSS, and JavaScript** and showcases a simple, responsive website design.

## Features

- Home page with flower shop introduction
- Multiple pages for products, about, and contact
- Interactive elements using JavaScript 
- Styled with CSS for a modern, clean look
- Responsive layout for desktop and mobile
